# JobBridge

AI-powered career marketplace connecting talent with employers. Job seekers find jobs, build AI-optimized resumes, and access career tools. Recruiters post jobs with AI-generated descriptions and smart candidate matching.

## Features

- **Job Listings** — Browse, filter, and apply for jobs across industries
- **AI Resume Builder** — Tailor resumes and generate cover letters (subscription required)
- **Recruiter Dashboard** — Post jobs, AI write descriptions, score candidates, manage applications
- **Service Providers** — Hire professionals in design, development, marketing, finance, and more
- **Gamification** — Career Match memory game with streak rewards
- **AI Assistant** — Intelligent chatbot answering questions about the platform
- **Admin Console** — Stealth admin panel for platform moderation and monitoring

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Backend:** Node.js, Express, SQLite
- **Payments:** Paystack
- **AI:** OpenAI GPT-4o-mini
- **3D Animations:** CSS transforms, IntersectionObserver
